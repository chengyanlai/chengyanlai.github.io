布尿布採買與開箱

Mon 16 April 2012
Yao-An Chan


這是目前幫兒子買的12件布尿布大集合。

由上至下、左至右的品牌如下：
  1. gDiapers x2
  2. Bonnibuns x2
  3. Mommy's Touch
  4. Rumparooz
  5. Blueberry x2 (不同款)
  6. bumGenius
  7. GroVia
  8. Thirsties x2 
總共8個牌子，不過因為兒子下個月才要退房，所以使用心得還要等一陣子吧。


在美國買布尿布有超多選擇，之前在採買的時候，在Amazon跟布尿布的官網看了老半天，但有的牌子的布尿布在網路上一直都只有正面照，內部長怎樣都看不到，不過我還是秉持著冒險精神買了很多品牌回來嘗試，也拍了很多照片，希望這些資訊對想採買布尿布的人有幫助。

一件式（All-in-one）跟口袋型（Pocket）都有附自己的吸水墊，這兩種都是用一次就要整件換掉。只有（Hybrid）傳統款因為只有外面的尿布兜，所以需要自行另外購買裡面的吸水墊。因為想說用該品牌自己出的吸水墊大小應該會比較合，秉持嘗試的心態，Hybrid那三牌我們都有買他們自己品牌出的吸水墊，也另外再買了一般要先折的布（prefolds）。

以上除了在babysteal買的兩件Bonnibuns另外付了快5塊美金運費之外，其他在amazon.com買的都是免運費、免稅，大部分還外加使用amazon的 baby registry 10% off，只有一兩件是買特價不是買9折。 

大部分都是6-8磅就可以開始穿，其中Thirsties跟gDiapers有分尺寸，所以我們買s的，想說買幾件比較小尺寸的看看，怕不分尺寸的剛開始穿會很大包。其他都是單一尺寸，可以用扣子改變大小一直穿到30~40磅。

以下附上各家的官網連結以及我目前對實品觀察的心得：

  1. gDiapers
    1. http://www.gdiapers.com/
    2. Size：newborn: 6-10lbs, s: 8-14lbs, m: 13-28lbs, L: 26-36lbs, XL: 34+
    3. gDiapers的官網做的很不錯，還有影片教學。他們家尿布兜的特色就是中間有一個用扣子扣上去的防水層，中間可以塞一般布的吸水墊，也可以塞他們家出的可以沖進馬桶的拋棄式吸水墊（gRefills），這樣就只要洗外面的防水層就好（如果布兜的地方沒沾到的話）。不過拋棄式的吸水墊並不便宜，價格就跟一般紙尿褲差不多了，沒有省到錢，只能說還是比紙尿褲環保一點。
    4. 整體而言，gDiaper算是比較貴的，不僅尺寸只能穿8-14磅，還要另外準備吸水墊，只能希望它貴得有價值啦！
    5. 我們也有買他們自家出的gCloth insert，想說這樣大小剛好，省去折來折去的麻煩。
  2. bumGenius
    1. http://www.bumgenius.com/
    2. 傳說中的BG 4.0口袋型，有附兩塊吸水墊。跟其他單一尺寸的比起來算是中規中矩的一款。
  3. Mommy's Touch
    1. http://www.mommystouch.com/
    2. All-in-one，長的吸水墊上方那邊是縫死不能分開的，靠前面那塊短的吸水墊有扣子可以拆下來。
    3. 正在特價，所以算起來是最便宜的。
  4. Rumparooz
    1. http://rumparooz.com/ 
    2. 前面扣子很多，似乎因為如此，可以使用的尺寸範圍是最大的，可以從6磅穿到40磅。
    3. 跟其他單一尺寸的比起來，Rumparooz內層多做了很立體的側邊。
    4. 是單一尺寸裡面最貴的，但實品整體感覺起來用料很好，看來是一分錢一分貨啊！希望實際上也很好用。
  5. Blueberry
    1. http://www.blueberrydiapers.com/Pocket-Diapers_2
    2. 買了兩款，Minky Diaper Snaps（迷彩那件）表面是像是絨布材質毛毛的，感覺很適合冬天穿。Deluxe Diaper Snaps（乳牛花紋那件）表面是比較類似防水材質的感覺。不知道是不是因為表面的材質而有價差，沒記錯的話，尿布裡面兩款是長得一樣的，附的吸水墊也是一樣。
  6. Bonnibuns 
    1. http://www.bonnibuns.com.au/
    2. 在baby steal網站上買的，號稱45% off ，原價28美金！這原價有夠貴。
    3. 號稱吸水墊拿掉可當寶寶泳褲，表面材質感覺是蠻快乾的。
    4. 吸水墊摸起來很舒服，說明書說不建議浸泡以免破壞竹纖維。
  7. GroVia
    1. https://www.gro-via.com/products.html 
    2. 這款是不分尺寸裡我唯一買魔鬼沾而不是扣子的，想說用看看，而且配色很可愛。XD
    3. 尿布兜的內襯是網狀布，感覺很透氣。
    4. 有扣子可以配他們自家出的不算便宜的吸水墊，不過吸水墊側邊做得很立體防漏的樣子，中間還可以分開，應該可以很快晾乾，感覺很厲害。
  8. Thirsties
    1. http://www.thirstiesbaby.com/products/cloth-diapers/duo-wrap/
    2. 兩種Size：6-18lbs, 另有18-40lbs
    3. 感覺非常快乾的防水材質。
    4. 跟gDiaper比起來，這款尿布兜內側有多做了彈性的側邊，gDiaper就是一件棉褲而已。
另外我們還買了Flushable Diaper Liner，長得有點像介於面紙跟濕紙巾中間的材質，不過它是乾的，擺在吸水墊上，遇到寶寶便便，就可以直接把Liner丟進馬桶，快速清掉便便。在Amazon上看別人的使用心得，好像非常方便，

洗了好幾次布尿布，萬事具備，希望之後會如預期的好用囉！

補充產地：
  • bumGenius, Mommy's Touch, Blueberry, Thirsties 這四牌made in U.S.A. 。
  • gDiapers, Rumparooz, GroVia這三牌是made in China。
  • Bonnibuns我找不到標籤 @_@
