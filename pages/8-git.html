Git

Here is my favorite Git command and some git related posts.
Some prior setups needed, please see Git Setup.

Commands

Super basic ...

$ git ci [--amend] # Don't amend published commit!
$ git add [-u] [-A]
$ git st # status
$ git bh # branch
$ git remote add <remote> <url>

Tags

# To add
$ git tag -a <tag name> {SHA}
# To delete
$ git tag -d <tag name>
# To push all tags
$ git push origin --tags
# To list all tags
$ git tag
# pattern?
$ git tag -l <pattern>
# To check out tag into new branch
$ git checkout -b <new_branch_name> <tag name>

Branch

# To create
$ git branch <branch name>
# To delete
$ git branch -d <branch name>
# To update
$ git rebase master

Rebase/Merge tool

$ git mergetool

Stash

$ git stash
$ git stash pop

See log ()

$ git slog # git log --oneline -20
# or
$ git last # git log -1 HEAD
# on specific file
$ git log -p <filename>

Reset

$ git reset --soft HEAD^{some number}
$ git reset --hard HEAD # remove all changes

List files under version control

$ git ls-files
